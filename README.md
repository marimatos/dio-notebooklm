# Mini guia de estudos - Power BI para análise de dados
 Contexto e Objetivos: Acesso fácil a fundamentos e conceitos de ferramentas do Power BI

## **Curadoria de Fontes:**
Utilizado fontes como oracle e microsoft  

[Microsoft - Fundamentos Power BI](https://learn.microsoft.com/pt-br/power-bi/fundamentals/power-bi-overview)  
[Microsoft - Power BI](https://learn.microsoft.com/pt-br/power-bi/)  
[Microsoft - Análise de Dados](https://learn.microsoft.com/pt-br/training/paths/data-analytics-microsoft/)  
[Oracle - Análise de Dados](https://www.oracle.com/br/analytics/data-analytics/)

## **Engenharia de Prompts e "Cicatrizes":**
No exemplo: "como posso usar o power bi para análise de dados?" - prompt um pouco vago que trouxe uma resposta longa, mas listando os principais fundamentos do Power BI.  
 Ao refazer o prompt com ajustes de output como: especificação de tom, limite de linhas, tipo de pessoa a ser direcionada a resposta e etc, obtive um retorno satisfatório e objetivo.  

 ## **Miniguia de Estudo:**
 * **Resumos estruturados**: conceitos básicos de Power BI, criação de dashboards, modelagem de dados e análise de indicadores utilizando o NotebookLM como apoio aos estudos.
 *  **Glossário**: Dashboard (painel visual de indicadores), DAX (linguagem de fórmulas do Power BI), ETL (extração, transformação e carga de dados), KPI (indicador de desempenho) e Data Model (modelo de dados).  
 * **Resultado final**: organização centralizada do conteúdo, facilitando consultas rápidas, revisões e aprendizado contínuo sobre Power BI.   
 * **Prompts**:  
         "Como o Power BI garante a segurança dos dados?"  
         "Como funciona a linguagem DAX para criar cálculos?"  
         "Como criar uma medida paara somar a coluna de volume da tabela xyz?"
